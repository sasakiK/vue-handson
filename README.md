# Vue.js hands-on

## やること

1. :one: Vueプロジェクトを作成
2. :two: 新しいComponentを作成
3. :three: apiを呼んでみる
4. :four: inputを反映してapiを呼んでみる

## 準備

### 必要なもの
    - vscode
    - terminal

### インストールしておくもの

- brewを一旦更新
```
brew upgrade --cleanup
```

- nodeをinstall
```
brew install node
```

- vue cliをinstall
```
npm install -g @vue/cli
```
